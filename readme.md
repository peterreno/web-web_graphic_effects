# 贪吃蛇前端小游戏

一个零依赖、即开即玩的贪吃蛇小游戏。直接打开 `index.html` 即可离线游玩，也可以使用本仓库提供的本地服务脚本启动一个静态网站。

## 功能亮点

- 🎮 经典 20×20 网格玩法，支持键盘方向键与 WASD
- ⚡ 逐步加速的蛇身，挑战更高难度
- 🧠 自动保存的本地最高分
- ⏯️ 开始、暂停、重新开始等实用控件
- 🌈 复古赛博风格 UI、柔和的网格和霓虹效果
- 🔊 内置音效，无需额外资源

## 本地运行

> 项目不依赖任何 npm 包，只需要浏览器即可体验。

### 方式一：直接打开
1. 克隆或下载仓库代码。
2. 用任意现代浏览器打开根目录下的 `index.html`。

### 方式二：通过静态服务器（可选）
如果希望通过 `http://localhost` 的方式体验，可在根目录执行：

```bash
npm run start
```

该命令会使用 Python 内置的 `http.server` 启动一个 4173 端口的静态服务器，按下 `Ctrl+C` 可停止服务。

## 操作说明

- `← ↑ → ↓` / `W A S D`：控制移动方向
- `Space`：开始或暂停
- 「暂停 / 继续」「重新开始」按钮提供鼠标操作入口
